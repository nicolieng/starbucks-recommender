# ☕ Starbucks Coffee Recommender


## 🛠️ Tech Stack
- Python
- pandas, numpy, scikit-learn, OpenAI (gpt-4o)
- CLI-based user input
- Modular design with src/ folder structure

## 🚀 How to Run
Clone the repo:
```
git clone https://github.com/your-username/starbucks-recommender.git
```


Running the program:
```
python main.py
```
